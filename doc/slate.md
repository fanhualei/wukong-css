# 2. 快速开始



## 2.1 新建React工程

你需要在你的机器上安装 [Node >= 8.10 和 npm >= 5.6](https://nodejs.org/en/)。要创建项目，请执行：

```shell
npx create-react-app slate
cd slate
npm start
```
